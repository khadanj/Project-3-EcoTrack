# EcoTrack: Personal Carbon & Habit Tracker

## Project Description

EcoTrack is a full stack web application that helps everyday people understand and reduce their personal carbon footprint. Users log the activities they do every day — driving to work, eating a meal, running the heat — and the app turns those logs into a concrete "Daily Footprint" score they can actually act on.

Users can also set monthly reduction goals, like cutting down on meat or taking public transit more often, and track their progress over time.

### Tech Stack:
- Frontend: React (Hooks) with component-specific CSS
- Backend: Node + Express (ES Modules)
- Database: MongoDB


## User Personas

**Lisa, 23, College Student** – Drives, takes transit, or flies regularly and wants to understand how her travel habits contribute to her personal carbon footprint.

**Darius, 33, Wants to do more for the environment** – Cares about sustainability but has never tracked his impact before and is looking for a simple way to start logging daily activities and see his footprint score.

**Pretti, 27, Environmentalist** – Already aware of her habits and wants to set concrete monthly reduction goals and track her progress over time to hold herself accountable.


## User Stories

### Tamiyoo Desir: Activity Log Manager

1. As a user, I want to log a new activity (e.g., "30-mile drive") so that I can track my daily emissions.
2. As a user, I want to view a list of all my logged activities filtered by date so that I can see my history.
3. As a user, I want to edit an existing log entry so that I can fix an accidentally entered wrong distance or category.
4. As a user, I want to remove a log entry so that I can delete entries that were created in error.

### James Khadan: Goals & Milestone Manager

1. As a user, I want to set a monthly reduction goal (e.g., "Reduce meat consumption by 20%") so that I can stay motivated.
2. As a user, I want to see my progress toward my active goals so that I can track how my activity data measures up.
3. As a user, I want to adjust the target values of my goals so that I can keep them relevant as my lifestyle changes.
4. As a user, I want to delete a goal so that I can remove ones that are no longer relevant to my journey.
